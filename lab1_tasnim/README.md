 Project Description
This project has two parts:
1. Python script that evaluates student grades from a CSV file
2. Bash script that archives grade files with timestamps

 Files in this Repository
- grade-evaluator.py - Python program that processes grades
- organizer.sh - Bash script that archives grades.csv
- grades.csv - Input file containing student grades
- organizer.log - Log file tracking archive operations
